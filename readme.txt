# Imo-Match Prototype

Prototype d’application de mise en relation immobilière (acheteurs ↔ vendeurs)  
basé sur la collecte de micro-données et un moteur de matching prédictif.

## Déploiement
Application en ligne : [imo-match.streamlit.app](https://imo-match.streamlit.app/)

## Fonctionnalités
- Formulaire acheteur : critères (budget, localisation, type de bien…)
- Formulaire vendeur : caractéristiques du bien
- Base SQLite intégrée
- Algorithme de matching avec score de compatibilité
- Affichage des correspondances en temps réel
